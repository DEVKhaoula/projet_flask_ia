# Mini Site Flask IA

Application web développée avec Flask permettant d’interagir avec un modèle d’intelligence artificielle via Ollama.

## Fonctionnalités

- Saisie d’un prompt utilisateur
- Envoi du prompt à un modèle IA
- Génération et affichage de la réponse
- Interface simple avec Flask et HTML

## Technologies utilisées

- Python
- Flask
- HTML/CSS
- Ollama

## Structure du projet

```bash
projet_flask_ia/
│
├── app.py
├── templates/
│   └── index.html
└── README.md
```

## Installation

### 1. Cloner le projet

```bash
git clone https://github.com/DEVKhaoula/projet_flask_ia.git
```

### 2. Accéder au dossier

```bash
cd projet_flask_ia
```

### 3. Installer Flask

```bash
pip install flask
```

### 4. Lancer l’application

```bash
python app.py
```

## Accès à l’application

Ouvrir dans le navigateur :

```bash
http://127.0.0.1:5000
```
